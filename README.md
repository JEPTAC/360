# Globo 3D de San Pedro con cartografía integrada

Esta versión usa la cartografía enviada como base visual integrada al globo 3D.

Incluye:
- cartografía municipal integrada en el render;
- casco urbano con iglesia y plaza;
- montañas posteriores y zona rural;
- visor HTML explorable;
- versión premium para landing page.
